Moving towards a goal

Consider the board on the right, where your bot wants to move to a target position. Each time you refresh this page the bot and target positions will change.

How can the bot reach the target?

  • We have four directions that we can consider: above, below, leftwards and rightwards.
  • The idea is that we will choose the move whose next position is closest to the target.
  • For determining how close a position \((x_1,y_1)\) is to another position \((x_2,y_2)\), we can remember school arithmetic and use the Euclidian distance: $$d = \sqrt{{|x_1-x_2|}^2 + {|y_1-y_2|}^2}$$
  • The function distanceFrom calculates how far a position is from the target. To discourage the bot from trying to move into walls, we set a very high distance value for walls.
  • Since we want to compare distances but select moves, we first compute the distances, and then compare them to select the closest move.
  • The moveTowards function, corresponding to this bot, is also available in the bot interface.
