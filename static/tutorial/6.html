Finding the closest power-up

Consider the board on the right, where your bot and various power-ups are randomly positioned on the board. Each time you refresh this page their positions will change.

You may have noticed that the previous bot does not always find the closest power-up. This is because we return the first power-up that we find when scanning the board.

How can the bot pick up the closest power-up?

  • The first note is that we don't need to find a single power-up, but all of them, and then choose the closest. To make this clearer, we will do it separately.
  • The findPowerups function finds all power-ups in the board. Unlike the previous bot, we now keep a List of all positions that contain power-ups.
  • We then use the closestTo function to go over all found power-up positions and choose the one that is closer to our bot's position. Note that we can iterate over lists as we did before for the board.
  • The findPowerups and the closestTo functions are also available in the bot interface.
