Finding a power-up

Consider the board on the right, where your bot and various power-ups are randomly positioned on the board. Each time you refresh this page their positions will change.

How can the bot pick up all the power-ups?

  • We only need to think about a power-up at a time. To find a power-up, we need to scan the board. In programming, this is usually done using iteration. The function iterateBoard goes over all the cells in the board: the idea is that we can define an initial value, and update it as we see each cell; the result is the final value after iterating over all the cells.
  • Since we want to find a power-up, our value will be a power-up's position. But when we start iterating, we still haven't found any power-up. For that purpose, we can use a Maybe to express if we have found Nothing or Just a position.
  • In our update function, we return the previously found power-up if it is a Just, otherwise we look into the current cell to find a power-up. Note that the case statement allows us to enumerate all the options.
